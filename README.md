# Earthquake-Visualization

A computer graphics simulation that displays over one hundred years of earthquake data across the globe. Toggles between several layers to visualize the map in two and three dimensions, as well as the map's vertex, index, and texture mappings. Earthquake and map vertices use linear interpolation to smoothly transition between dimensions, and display earthquake magnitudes. Created via GopherGfx as part of an assignment for CSci 4611.
